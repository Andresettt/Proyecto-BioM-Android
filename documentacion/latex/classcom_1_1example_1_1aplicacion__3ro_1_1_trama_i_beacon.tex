\doxysubsection{Descripción detallada}
Clase que representa y procesa una trama i\+Beacon. 

Esta clase se encarga de interpretar los datos de una trama i\+Beacon, extrayendo información como el UUID, el major, el minor, y la potencia de transmisión (Tx\+Power), entre otros campos.
