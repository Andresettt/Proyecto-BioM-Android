\doxysubsection{Descripción detallada}
Clase de utilidades para conversiones de datos y manipulación de UUIDs. 

Esta clase incluye métodos para convertir cadenas a bytes, UUIDs a cadenas, conversión de bytes a tipos primitivos como int y long, y otros tipos de conversiones útiles.
