Descripción detallada

Interfaz para definir las llamadas HTTP de la API.

Esta interfaz define el método para enviar las mediciones al servidor utilizando una solicitud HTTP POST. Retrofit se encargará de crear la implementación de esta interfaz en tiempo de ejecución.
